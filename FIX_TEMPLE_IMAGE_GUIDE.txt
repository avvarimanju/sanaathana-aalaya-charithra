TEMPLE IMAGE FIX - QUICK GUIDE
================================

PROBLEM: Temple cards showing placeholder icon 🏛️ instead of real photos

SOLUTION: Add image URL to your temple

STEPS:
------

1. FIND A TEMPLE IMAGE ONLINE
   - Search Google Images for "Vidurashwatha Temple"
   - Right-click on the image → "Copy image address"
   - Example URL format: https://example.com/temple-image.jpg

2. EDIT YOUR TEMPLE IN ADMIN PORTAL
   - Go to admin portal: http://localhost:3001
   - Click on "Vidurashwatha Temple" card
   - Click the Edit button (pencil icon)
   - Scroll to "Image URL" field
   - Paste the image URL
   - Click "Update Temple"

3. REFRESH MOBILE APP
   - The temple card will now show the real photo!

ALTERNATIVE: Use a Free Image Hosting Service
----------------------------------------------

If you have a photo on your computer:

1. Upload to Imgur (https://imgur.com)
   - Click "New post"
   - Upload your image
   - Right-click uploaded image → "Copy image address"

2. Or use Google Photos
   - Upload image
   - Open image
   - Right-click → "Copy image address"

3. Paste the URL in admin portal as described above

RESTART REQUIRED:
-----------------
If your local backend server is running, restart it to pick up the changes:

1. Stop the server (Ctrl+C)
2. Run: npm run start:local-backend
   (or whatever command you use to start the local server)

Then edit your temple and add the image URL!
