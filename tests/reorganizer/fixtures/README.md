# Test Fixtures

This directory contains sample project structures for testing the reorganization system.

## Structure

- `sample-project/` - A sample project structure with documentation files in the root
- `expected-output/` - The expected structure after reorganization

## Usage

Tests will use these fixtures to verify that the reorganization system correctly:
- Identifies files for relocation
- Moves files to appropriate subdirectories
- Updates file references
- Validates the final structure
