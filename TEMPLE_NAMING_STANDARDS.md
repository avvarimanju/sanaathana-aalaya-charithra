# Temple Naming Standards

Guidelines for using official and authentic temple names.

**Last Updated**: March 3, 2026  
**Status**: Naming Convention Standards

---

## Core Principle

**ALWAYS use the official temple name from the temple's official website or signboard.**

Do NOT use:
- ❌ Popular names
- ❌ Anglicized names
- ❌ Shortened names
- ❌ Tourist names

---

## Correct vs Incorrect Examples

### Example 1: Tirupati Temple

**WRONG**: "Tirupati Balaji Temple"  
**CORRECT**: "Sri Venkateswara Swamy Temple" or "Sri Venkateswara Swamy Vari Devasthanam"

**Source**: Official TTD (Tirumala Tirupati Devasthanams) website  
**Why**: "Tirupati Balaji" is a popular name, not the official name

---

### Example 2: Golden Temple

**WRONG**: "Golden Temple"  
**CORRECT**: "Sri Harmandir Sahib" or "Darbar Sahib"

**Source**: SGPC (Shiromani Gurdwara Parbandhak Committee) official website  
**Why**: "Golden Temple" is a tourist name, not the official Sikh name

---

### Example 3: Meenakshi Temple

**WRONG**: "Meenakshi Temple"  
**CORRECT**: "Arulmigu Meenakshi Sundareswarar Temple"

**Source**: Temple official website and HR&CE Tamil Nadu  
**Why**: Full official name includes both deities

---

### Example 4: Brihadeeswarar Temple

**WRONG**: "Big Temple" or "Thanjavur Temple"  
**CORRECT**: "Peruvudaiyar Kovil" or "Brihadeeswarar Temple"

**Source**: ASI (Archaeological Survey of India) and local signage  
**Why**: "Big Temple" is a colloquial name

---

### Example 5: Jagannath Temple

**WRONG**: "Jagannath Temple"  
**CORRECT**: "Shree Jagannath Temple" or "Sri Jagannath Mandir"

**Source**: Shree Jagannath Temple Administration, Puri  
**Why**: Official name includes honorific "Shree" or "Sri"

---

## Naming Format Guidelines

### 1. Include Honorifics

Most temples have official honorifics:

**Common Honorifics**:
- Sri / Shri / Shree (Sanskrit)
- Arulmigu (Tamil - "blessed by grace")
- Periya (Tamil - "great")
- Maha (Sanskrit - "great")

**Examples**:
- Sri Venkateswara Swamy Temple
- Arulmigu Meenakshi Sundareswarar Temple
- Shree Jagannath Temple
- Maha Kumbhabhishekam

---

### 2. Include Full Deity Names

Use complete deity names, not shortened versions:

**WRONG**: "Shiva Temple"  
**CORRECT**: "Sri Kashi Vishwanath Temple"

**WRONG**: "Krishna Temple"  
**CORRECT**: "Sri Krishna Matha" or "Sri Udupi Krishna Temple"

---

### 3. Regional Language Names

Prefer the local language name when available:

**Tamil Nadu**:
- Use Tamil names: "Arulmigu", "Kovil"
- Example: "Arulmigu Kapaleeswarar Temple"

**Andhra Pradesh/Telangana**:
- Use Telugu names: "Sri", "Swamy", "Devasthanam"
- Example: "Sri Venkateswara Swamy Vari Devasthanam"

**Karnataka**:
- Use Kannada names: "Sri", "Swamy", "Matha"
- Example: "Sri Krishna Matha, Udupi"

**Kerala**:
- Use Malayalam names: "Sri", "Kshetram", "Devaswom"
- Example: "Sri Padmanabhaswamy Temple"

## How to Find Official Names

### 1. Temple Official Website

**Best Source**: Always check the temple's official website first

**Examples**:
- Tirumala Tirupati: https://www.tirumala.org
- Sabarimala: https://sabarimala.kerala.gov.in
- Jagannath Puri: https://jagannath.nic.in
- Somnath: https://somnath.org

---

### 2. Temple Management Boards

**Tamil Nadu**: HR&CE (Hindu Religious & Charitable Endowments)
- Website: https://www.hrce.tn.gov.in
- Lists official temple names

**Andhra Pradesh**: Endowments Department
- Website: https://www.apendowments.gov.in

**Karnataka**: Muzrai Department
- Website: https://muzrai.karnataka.gov.in

**Kerala**: Devaswom Boards
- Travancore Devaswom Board: https://www.tdb.kerala.gov.in
- Cochin Devaswom Board: https://www.cochindevaswomboard.org
- Malabar Devaswom Board: https://malabardevaswomboard.org
- Guruvayur Devaswom: https://guruvayurdevaswom.org
- Sabarimala Temple: https://sabarimala.kerala.gov.in

---

### 3. Archaeological Survey of India (ASI)

For heritage temples:
- Website: https://asi.nic.in
- Uses official historical names

---

### 4. Temple Signboards

If visiting in person:
- Check the main entrance signboard
- Look for inscriptions
- Check official pamphlets

---

### 5. Government Tourism Websites

State tourism departments usually use official names:
- Tamil Nadu Tourism
- Andhra Pradesh Tourism
- Karnataka Tourism
- Kerala Tourism

## Common Mistakes to Avoid

### 1. Using City Names as Temple Names

**WRONG**: "Tirupati Temple"  
**CORRECT**: "Sri Venkateswara Swamy Temple" (located in Tirupati)

**WRONG**: "Madurai Temple"  
**CORRECT**: "Arulmigu Meenakshi Sundareswarar Temple" (located in Madurai)

---

### 2. Using Deity Names Only

**WRONG**: "Shiva Temple"  
**CORRECT**: "Sri Kashi Vishwanath Temple"

**WRONG**: "Vishnu Temple"  
**CORRECT**: "Sri Ranganathaswamy Temple"

---

### 3. Omitting Honorifics

**WRONG**: "Venkateswara Temple"  
**CORRECT**: "Sri Venkateswara Swamy Temple"

**WRONG**: "Meenakshi Temple"  
**CORRECT**: "Arulmigu Meenakshi Sundareswarar Temple"

---

### 4. Using English Translations

**WRONG**: "Big Temple" (for Brihadeeswarar)  
**CORRECT**: "Brihadeeswarar Temple" or "Peruvudaiyar Kovil"

**WRONG**: "Shore Temple" (for Mahabalipuram)  
**CORRECT**: "Mahabalipuram Shore Temple" or "Kadal Alaya"

    # 1. Check temple official website

        official_name = fetch_from_management_board(state, city)
    
    # 3. Check ASI database
    if not official_name:
        official_name = fetch_from_asi(temple_id)
    
    # 4. Manual verification required
    if not official_name:
        log_for_manual_verification(temple_id)
    
    return official_name
