### Option 1: Stop and Start (Recommended)
1. Find the terminal running the backend (Terminal 26 or similar)
2. Press `Ctrl+C` to stop the server
3. Run: `npm run start:local` or `node src/local-server/server.js`

### Option 2: Use the restart script
```powershell
cd Sanaathana-Aalaya-Charithra
.\scripts\start-local-backend.ps1
```
