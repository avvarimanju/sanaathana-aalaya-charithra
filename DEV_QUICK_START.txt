🔥 START ALL DEV SERVERS (DO THIS ONCE)
═══════════════════════════════════════════════════════════════════

  .\scripts\dev-watch-all.ps1

This opens 3 windows with hot-reload enabled:
  • Backend:      http://localhost:4000  (auto-restarts)
  • Admin Portal: http://localhost:5173  (instant updates)
  • Mobile App:   Expo Dev Tools          (fast refresh)


✨ HOW IT WORKS
═══════════════════════════════════════════════════════════════════

1. Edit any file
2. Save (Ctrl+S)
3. Changes appear AUTOMATICALLY
4. No restart needed!

Admin Portal:  Changes appear in < 1 second
Mobile App:    Changes appear in < 2 seconds  
Backend:       Auto-restarts in 2-3 seconds


💾 COMMIT YOUR CHANGES
═══════════════════════════════════════════════════════════════════

When ready to save your work:

  .\scripts\auto-commit-dev.ps1 -Message "Your changes"


🎯 TYPICAL WORKFLOW
═══════════════════════════════════════════════════════════════════

Morning:
  .\scripts\dev-watch-all.ps1        # Start servers

During Day:
  1. Edit code
  2. Save file
  3. See changes instantly
  4. Repeat

Periodic:
  .\scripts\auto-commit-dev.ps1      # Commit progress

End of Day:
  .\scripts\auto-commit-dev.ps1      # Final commit
  Close terminal windows             # Stop servers


❌ YOU DON'T NEED TO:
═══════════════════════════════════════════════════════════════════

  ❌ Run npm run build
  ❌ Restart servers manually
  ❌ Deploy to staging/production
  ❌ Wait for compilation
  ❌ Refresh browser manually (Admin Portal)
  ❌ Reload app manually (Mobile App)


✅ WHAT HAPPENS AUTOMATICALLY:
═══════════════════════════════════════════════════════════════════

  ✅ File changes detected
  ✅ Code recompiled
  ✅ Browser/app updated
  ✅ State preserved (when possible)


🚨 IF SOMETHING BREAKS:
═══════════════════════════════════════════════════════════════════

Hard Refresh Browser:
  Ctrl + Shift + R

Reload Mobile App:
  Shake device → "Reload"

Restart All Servers:
  Close windows → Run dev-watch-all.ps1 again


📊 WHEN TO ACTUALLY DEPLOY:
═══════════════════════════════════════════════════════════════════

Staging (for team review):
  .\scripts\auto-deploy-all.ps1 -Environment staging

Production (for users):
  .\scripts\auto-deploy-all.ps1 -Environment production


💡 PRO TIPS:
═══════════════════════════════════════════════════════════════════

• Enable Auto Save in VS Code: File → Auto Save
• Keep terminal windows visible to see errors
• Commit after completing each feature
• Use Ctrl+C to stop individual servers
