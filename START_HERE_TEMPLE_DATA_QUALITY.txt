MANDATORY RULES GOING FORWARD:

1. ✅ ALWAYS use official temple name from temple website
   Example: "Sri Venkateswara Swamy Temple" NOT "Tirupati Balaji Temple"

2. ✅ ALWAYS verify links have NO SPACES
   Example: https://www.tdb.kerala.gov.in NOT https://www.travancore devaswomboard.org

3. ✅ ALWAYS use HTTPS protocol
   Example: https://www.tirumala.org NOT http://tirumala.org

4. ✅ ALWAYS test links in browser before adding
   - Open in browser
   - Verify it loads
   - Confirm it's official website

5. ✅ ALWAYS include honorifics
   Example: "Sri", "Arulmigu", "Shree", "Shri"

6. ✅ ALWAYS add alternate names for search
   Store popular names in alternateNames array

7. ✅ ALWAYS document official source URL
   Add officialSource field with verified link

8. ✅ ALWAYS use standardized temple ID format
   Format: STATE-CODE_DISTRICT-CODE_CITY-CODE_temple-name
