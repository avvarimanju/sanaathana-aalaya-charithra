### If Using Cloudflare (Recommended)

**Advantage**: Cloudflare provides both domain registration AND DNS hosting, so you can skip AWS Route 53 and save $6/year!

**Updated Cost**:
```
Domain (Cloudflare):     $9.77/year
Route 53:                $0.00 (not needed!)
S3 Storage:              $0.12/year
CloudFront:              $0.00 (free)
SSL:                     $0.00 (free)
────────────────────────────────
TOTAL:                   $9.89/year ($0.82/month)
```

**Setup**:

4. Skip Route 53 entirely!
