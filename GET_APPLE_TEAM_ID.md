## Steps to Get Team ID

### Option 1: Direct Link (Fastest!)

Go directly to:
```
https://developer.apple.com/account/#/membership
```

This takes you straight to the Membership Details page.

### Option 2: Navigate Manually

1. You're already on: https://developer.apple.com/account
2. Look at the **left sidebar**
3. Click **"Membership"**
4. Scroll down to find **"Team ID"**

### What You'll See

On the Membership page, look for:
- **Membership Information** section
- **Team ID** field with a 10-character code

Example formats:
- `A1B2C3D4E5`
- `ABCD123456`
- `1A2BC34DE`

### Copy the Team ID

Copy the entire 10-character Team ID exactly as shown.

## Alternative: If You Don't See "Membership"

If you don't see a "Membership" link, try:

1. **Look for "Account" or "Overview"** in the sidebar
2. **Check the top-right corner** - your Team ID might be displayed there
3. **Look for "Certificates, Identifiers & Profiles"** - Team ID is often shown there

---

## What If You Haven't Enrolled Yet?

If you see a message like "Join the Apple Developer Program":

1. You need to enroll first ($99/year)
2. Click "Enroll today" button
3. Follow the enrollment process (takes 1-2 days for approval)
4. Come back after approval to get your Team ID
