## Using Your Existing Repository (Recommended)

Since you already have the `Sanaathana-Aalaya-Charithra` repository on GitHub, we'll use that instead of creating a new one. This is simpler and keeps everything together.

### Steps to Enable GitHub Pages

**1. Push the privacy policy files to GitHub**

```powershell
cd Sanaathana-Aalaya-Charithra

# Add the new files
git add docs/privacy-policy.md docs/privacy-policy.html

# Commit
git commit -m "Add privacy policy for Play Store submission"

# Push to GitHub
git push origin main
```

**2. Enable GitHub Pages**

1. Go to your repository on GitHub: `https://github.com/avvarimanju/sanaathana-aalaya-charithra`
2. Click on "Settings" tab
3. Scroll down to "Pages" section (in the left sidebar under "Code and automation")
4. Under "Source", select:
   - Branch: `main`
   - Folder: `/docs`
5. Click "Save"

**3. Get your Privacy Policy URL**

After enabling GitHub Pages, your privacy policy will be available at:

```
https://[your-username].github.io/Sanaathana-Aalaya-Charithra/privacy-policy.html
```

For example, if your GitHub username is `avvarimanju`:
```
https://avvarimanju.github.io/Sanaathana-Aalaya-Charithra/privacy-policy.html
```

**4. Test the URL**

- Wait 2-3 minutes for GitHub Pages to deploy
- Open the URL in your browser
- Verify the privacy policy displays correctly with proper formatting

**Time**: 5-10 minutes

## Alternative Options (If You Don't Want to Use GitHub Pages)

### Option 2: GitHub Gist (Fastest - 5 minutes)

**Steps:**

1. Go to https://gist.github.com/
2. Click "New gist"
3. Filename: `privacy-policy.md`
4. Paste content from `docs/privacy-policy.md`
5. Create public gist
6. Copy the gist URL

**URL Format**: `https://gist.github.com/[username]/[gist-id]`

**Pros**: Fastest option  
**Cons**: Less professional URL

**Time**: 5 minutes

---

### Option 3: Google Sites (No GitHub Account Needed)

**Steps:**

1. Go to https://sites.google.com/
2. Click "Blank" to create new site
3. Site name: "Sanaathana Aalaya Charithra Privacy Policy"
4. Click on text area and paste privacy policy content
5. Click "Publish"
6. Choose URL: `sanaathana-privacy-policy`
7. Copy the published URL

**URL Format**: `https://sites.google.com/view/sanaathana-privacy-policy`

**Time**: 10 minutes

---

### Option 4: Netlify (Professional Option)

**Steps:**

1. Go to https://app.netlify.com/
2. Drag and drop the `docs/privacy-policy.html` file
3. Get your URL: `https://[random-name].netlify.app`
