WHAT'S NEW:
-----------
- Interactive SVG map of India with all 36 states/UTs

- Automatic temple filtering by selected state
- "Temples are being added..." message for empty states
- Full accessibility support
- 280+ unit tests

HOW TO TEST:
------------

1. START THE APP:
   cd mobile-app
   npm start

2. TEST ON WEB (Easiest):
   npm run web
   
   Then navigate:
   - Login (or Continue as Guest)
   - Select Language
   - **NEW: India Map Screen appears!**
   - Click a state on the map OR in the list
   - See filtered temples for that state

3. TEST ON MOBILE:
   npm run ios     (for iOS)
   npm run android (for Android)

4. RUN UNIT TESTS:
   npm test
   
   Should see 280+ tests passing!
