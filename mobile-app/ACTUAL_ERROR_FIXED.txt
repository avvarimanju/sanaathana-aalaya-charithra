THE REAL ERROR:
---------------
"Objects are not valid as a React child"

This error occurred in the browser console (not terminal!)


ROOT CAUSE:
-----------
Two screens were passing function references instead of calling them:

1. NotificationsScreen.tsx
   ❌ ListEmptyComponent={renderEmptyState}
   ✅ ListEmptyComponent={renderEmptyState()}

2. MyDefectsScreen.tsx
   ❌ ListEmptyComponent={renderEmptyState}
   ✅ ListEmptyComponent={renderEmptyState()}


WHAT WAS WRONG:
---------------
When you pass {renderEmptyState} without (), React receives
the function object itself, not the JSX it returns.

React then tries to render the function object as a child,
which causes the error.


THE FIX:
--------
Added () to call the functions:
- renderEmptyState() instead of renderEmptyState


FILES FIXED:
------------
✅ mobile-app/src/screens/NotificationsScreen.tsx
✅ mobile-app/src/screens/MyDefectsScreen.tsx
