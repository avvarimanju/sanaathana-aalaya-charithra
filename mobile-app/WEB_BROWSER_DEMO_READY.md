## Next Steps (2 Minutes)

### Step 1: Wait for Expo to Finish Starting

**Watch your terminal for:**
```

› Press w │ open web
› Press r │ reload app
› Press m │ toggle menu

```

**This usually takes 30-60 seconds.**

### Step 2: Open in Web Browser

**Once you see the menu above, press:**
```
w
```

**What happens:**
- Browser opens automatically
- URL: http://localhost:8081 (or similar)
- Your app loads in browser

**If browser doesn't open automatically:**
- Open Chrome or Edge manually
- Go to: http://localhost:8081

### Step 3: Enable Mobile View

**In the browser:**

1. **Press F12** (opens Developer Tools)
2. **Click device icon** 📱 (top-left of DevTools)
   - Or press: Ctrl + Shift + M
3. **Select device:** iPhone 12 Pro or Pixel 5
4. **Ensure:** Portrait orientation (vertical)

### Step 4: Make It Professional

**Hide DevTools:**
- Press F12 again

**Enable Fullscreen:**
- Press F11

**Done!** Your app now looks like a real phone! 📱

## Quick Commands Reference

### In Terminal (Expo Menu)
```
w - Open in web browser
a - Open in Android emulator
r - Reload app
c - Clear cache
q - Quit
```

### In Browser
```
F12              - Open/close DevTools
Ctrl+Shift+M     - Toggle mobile view
F11              - Fullscreen mode
Ctrl+0           - Reset zoom to 100%
Ctrl+R           - Refresh page
```

## Troubleshooting

### Issue: "App not loading"

**Solution:**
```bash
# In terminal, press: Ctrl+C to stop
# Then restart:
npm start -- --clear
```

### Issue: "Port already in use"

**Already handled!** We're using port 8082.

### Issue: "Layout looks wrong"

**Solution:**
```bash
# In browser:
Press Ctrl+0 (reset zoom to 100%)
Press F12 → Select correct device
Press Ctrl+R (refresh)
```
