GITHUB ACTIONS SETUP - QUICK START
===================================

STEP 1: Get Expo Token
-----------------------
1. Open: https://expo.dev/accounts/avvarimanju/settings/access-tokens
2. Click "Create Token"
3. Name: "GitHub Actions"
4. Click "Create"
5. COPY THE TOKEN (shown only once!)

STEP 2: Create GitHub Repository
---------------------------------
1. Go to: https://github.com/new
2. Repository name: sanaathana-mobile-app
3. Choose: Private
4. Click "Create repository"

STEP 3: Add Token to GitHub Secrets
------------------------------------
1. Go to your new repository
2. Settings > Secrets and variables > Actions
3. Click "New repository secret"
4. Name: EXPO_TOKEN
5. Value: (paste token from Step 1)
6. Click "Add secret"

STEP 4: Push Code to GitHub
----------------------------
Run these commands from PowerShell in this directory:

cd "C:\Users\avvar\OneDrive\LEARNING\Hack2Skill\Sanaathana-Aalaya-Charithra\mobile-app"

# Check git status
git status

# Add the workflow file
git add .github/workflows/eas-build.yml

# Commit
git commit -m "Add GitHub Actions workflow for EAS build"

# Add GitHub remote (replace YOUR_USERNAME with your GitHub username)
git remote add github https://github.com/YOUR_USERNAME/sanaathana-mobile-app.git

# Push to GitHub
git push github master

STEP 5: Trigger Build
----------------------
Option A: Automatic (already triggered by push)
Option B: Manual
  1. Go to: https://github.com/YOUR_USERNAME/sanaathana-mobile-app/actions
  2. Click "EAS Build" workflow
  3. Click "Run workflow"

STEP 6: Download AAB
--------------------
When build completes, download from:
https://expo.dev/accounts/avvarimanju/projects/sanaathana-aalaya-charithra-mobile/builds

DONE! The build runs on Linux, so no Windows permission issues!
