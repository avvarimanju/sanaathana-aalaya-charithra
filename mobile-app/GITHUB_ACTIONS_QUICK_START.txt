STEP 1: Get Expo Token
-----------------------
1. Open: https://expo.dev/accounts/avvarimanju/settings/access-tokens

------------------------------------
1. Go to: https://github.com/avvarimanju/sanaathana-aalaya-charithra/settings/secrets/actions
2. Click "New repository secret"
3. Name: EXPO_TOKEN
4. Value: (paste token from Step 1)
5. Click "Add secret"

--------------------------------------
Run these commands from PowerShell:
