  ❓ "Cannot find module" error?
     → Run: npx expo start --clear

  ❓ Port already in use?
     → Run: npx kill-port 8081
     → Or: npx expo start --web --port 8082

  ❓ Still not working?
     → Run: ./test-mobile-app.ps1
     → Check for specific errors
