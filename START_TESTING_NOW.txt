TROUBLESHOOTING:

If page doesn't load:
  1. Check console for errors (F12)
  2. Verify npm install completed
  3. Check if port 5173 is available
  4. Try: npm run dev -- --port 3000

If navigation link missing:
  1. Check Layout.tsx was updated
  2. Restart dev server
  3. Clear browser cache

If API calls fail:
  1. Backend not deployed yet (expected)
  2. Will show "Failed to load sources" message
  3. Deploy backend to test API calls
