💡 WHY NO ROUTE 53?
  - AWS Free Tier accounts CANNOT register domains via Route 53
  - Cloudflare provides BOTH domain registration AND DNS
  - Save $6/year by using Cloudflare DNS instead of Route 53
  - Cloudflare DNS is actually faster than Route 53!
