# AI Content Generation Prompt Template

Use this template when generating content about Hindu temples and artifacts for the Sanaathana Aalaya Charithra application.

## Prompt Template

```
You are a knowledgeable expert on Hindu temple architecture, history, and culture. Generate content about [ARTIFACT/TEMPLE NAME] following these strict guidelines:

SOURCES TO USE (in priority order):
1. Archaeological Survey of India (ASI) documentation
2. [Specific Temple Authority] official records (e.g., TTD for Tirumala)
3. State Archaeological Department records
4. Agama Shastras and traditional texts
5. Temple inscriptions (Shilashasanas)
6. Peer-reviewed academic research

CONTENT REQUIREMENTS:
- Focus on: [architecture/history/rituals/art/mythology]
- Language: [English/Hindi/Telugu/Kannada/Tamil]
- Tone: Respectful, educational, factual
- Length: [word count]
- Include: Historical dates, architectural features, cultural significance
- Avoid: Controversial topics, unverified claims, political content

SPECIFIC INSTRUCTIONS:
- Use proper Sanskrit/regional language terms with English translations
- Cite specific sources for historical facts
- Clearly distinguish between documented history and traditional legends
- Respect religious sentiments and traditional beliefs
- Include architectural terminology where relevant
- Mention UNESCO/ASI heritage status if applicable

FORMAT:
- Introduction (2-3 sentences)
- Main content (organized by subtopics)
- Key facts (bullet points)
- Sources used (list at end)

EXAMPLE OUTPUT STRUCTURE:
{
  "title": "Artifact/Temple Name",
  "introduction": "Brief overview...",
  "mainContent": "Detailed information...",
  "keyFacts": [
    "Fact 1 with source",
    "Fact 2 with source"
  ],
  "sources": [
    {
      "name": "Source name",
      "type": "primary/secondary",
      "reference": "Specific reference"
    }
  ],
  "language": "en",
  "lastUpdated": "2026-02-27"
}

Generate content now for: [SPECIFIC ARTIFACT/TEMPLE]
```

## Example Usage

### For Audio Guide Content

```
Generate an audio guide script (300-400 words) about the Hanging Pillar of Lepakshi Temple.

Use these sources:
- ASI documentation on Lepakshi Temple
- Andhra Pradesh State Archaeology records
- Agama Shastras on Vijayanagara architecture
- Temple inscriptions

Include:
- Historical period (Vijayanagara Empire, 16th century)
- Architectural marvel (suspended pillar phenomenon)
- Engineering techniques
- Cultural significance
- Visitor experience perspective

Tone: Engaging, informative, suitable for audio narration
Language: English (with Sanskrit terms translated)
```

### For Video Script Content

```
Generate a video script (2-3 minutes) about the Monolithic Nandi at Lepakshi.

Structure:
1. Opening (10 seconds): Hook about the massive sculpture
2. Historical context (30 seconds): Vijayanagara period, who built it
3. Architectural details (45 seconds): Size, carving technique, single granite rock
4. Cultural significance (30 seconds): Symbolism of Nandi, temple connection
5. Closing (15 seconds): Visitor information, preservation

Sources: ASI, AP Archaeology, traditional texts on Nandi symbolism
```

### For Infographic Content

```
Generate key data points for an infographic about Tirumala Temple.

Required information:
- Temple name and deity
- Location and state
- Historical period/age
- Architectural style
- Key features (3-5 points)
- UNESCO/Heritage status
- Annual visitors (if available)
- Best time to visit
- Unique facts (2-3)

Sources: TTD official records, ASI documentation
Format: Short phrases, numbers, bullet points
```

### For Q&A Chatbot Responses

```
Generate Q&A pairs about Sri Kalahasti Temple's Vayu Linga.

Common questions to address:
1. What is Vayu Linga?
2. Why is it called Vayu (Air) Linga?
3. What is the legend of spider, snake, and elephant?
4. What is Pancha Bhoota Sthalam?
5. When was the temple built?
6. What are the unique rituals here?

Requirements:
- Answers: 50-100 words each
- Cite sources for historical facts
- Explain Sanskrit terms
- Respectful tone
- Suitable for general audience

Sources: Temple authority records, Sthala Purana, ASI documentation
```

## Quality Checklist

Before using AI-generated content, verify:

- [ ] All historical facts have source citations
- [ ] Sanskrit/regional terms are properly translated
- [ ] Content respects religious sentiments
- [ ] No controversial or political statements
- [ ] Legends are clearly marked as traditional stories
- [ ] Architectural terms are accurate
- [ ] Dates and measurements are verified
- [ ] Content is appropriate for target audience
- [ ] Language is clear and engaging
- [ ] Sources are from approved list

## Content Review Process

1. **AI Generation**: Use prompt template with specific sources
2. **Fact Checking**: Verify against primary sources
3. **Expert Review**: Subject matter expert validation
4. **Temple Authority Approval**: For sensitive content
5. **Final Approval**: Content team sign-off
6. **Publication**: Add to database with source metadata

---

**Note**: Always prioritize accuracy and respect over creative embellishment. When in doubt, consult primary sources or temple authorities.
