📱
Mobile App
React Native cross-platform application for iOS and Android
  • Submit defect reports
  • View my defects
  • Track status updates
  • Receive notifications
💻
Admin Dashboard
React web application for administrators
  • Manage all defects

⚡
AWS API Gateway
Managed REST API with built-in security and scaling
  • REST API endpoints
  • JWT authentication
  • Rate limiting & throttling
  • Request validation
  • CORS configuration
⚙️ COMPUTE LAYER (AWS Lambda)
📝
Submit Defect
POST /defects - Create new defect reports
📋
Get User Defects
GET /defects/user/{userId} - List user's defects
🔍
Get Defect Details
GET /defects/{id} - Retrieve defect with history
✏️
Update Status
PUT /admin/defects/{id}/status - Change defect status
💬
Add Comment
POST /admin/defects/{id}/updates - Add status update
📊
Get All Defects
GET /admin/defects - Admin view all defects
🔔
Get Notifications
GET /notifications/user/{userId} - User notifications
✅
Mark Read
PUT /notifications/{id}/read - Mark as read
💾 DATA LAYER (DynamoDB)
🗂️
Defects Table
Primary defect data storage
  • PK: defectId
  • GSI: userId-createdAt-index
  • GSI: status-createdAt-index
  • On-demand capacity
  • Point-in-time recovery
📝
StatusUpdates Table
Status change history and comments
  • PK: updateId
  • GSI: defectId-timestamp-index
  • Admin comments
  • Status transitions
🔔
Notifications Table
User notification storage
  • PK: notificationId
  • GSI: userId-createdAt-index
  • TTL enabled (90 days)
  • Read status tracking
🔌 EXTERNAL SERVICES
📧
AWS SNS
Simple Notification Service for alerts
  • Email notifications

📊
CloudWatch
Monitoring and logging service
  • Lambda execution logs
  • Performance metrics
  • Error tracking
  • Custom alarms
📊 Request/Response Flow: User Submits Defect
1
User Action
User fills out defect form in mobile app and submits
2
HTTPS Request
Mobile app sends POST /defects to API Gateway
3
Authentication & Validation
API Gateway validates JWT token and rate limits
4
Lambda Invocation
Submit Defect Lambda function is triggered
5
Business Logic
Validate data with Zod, generate UUID, set status to "New"
6
Database Write
Store defect in DynamoDB Defects table
7
Logging
Log submission event to CloudWatch
8
Response
Return 201 Created with defectId to mobile app
