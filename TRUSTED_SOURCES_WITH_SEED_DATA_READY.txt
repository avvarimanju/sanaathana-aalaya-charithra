🔧 TECHNICAL DETAILS
═══════════════════════════════════════════════════════════════════════════════

Seed Data Location:
  data/trusted-sources-seed.json

Loader Code:
  src/local-server/trustedSourcesRoutes.ts

How It Works:
  1. Server starts
  2. Reads seed JSON file
  3. Loads all 45+ sources into memory
  4. Sources available immediately
  5. Admins can add/edit/delete as needed

Fallback:
  If seed file not found, loads 1 sample source

📝 ADDING MORE SOURCES
═══════════════════════════════════════════════════════════════════════════════

Option 1: Edit Seed File (Permanent)
  1. Edit data/trusted-sources-seed.json
  2. Add new source to "sources" array
  3. Restart backend server
  4. New source appears for everyone

Option 2: Via Admin Portal (Temporary)
  1. Click "Add New Source" button
  2. Fill in details
  3. Save
  4. Source added (but resets on server restart)

Recommendation: Use Option 1 for permanent sources
