## Link Verification Process

### Before Adding Any Link

1. **Test the URL** in a browser
2. **Verify it loads** without errors
3. **Check it's the official** website (not third-party)
4. **Confirm HTTPS** (secure connection)
5. **Document verification date**

### Link Format Standards

✅ **CORRECT**:
```
https://www.tirumala.org
```

❌ **WRONG** (Examples of invalid formats):
```
EXAMPLE 1: https://www.travancore devaswomboard.org  (space in URL)
EXAMPLE 2: [INVALID-PROTOCOL]://tirumala.org  (not using HTTPS)
EXAMPLE 3: tirumala.org  (missing protocol)
```

## Verification Checklist

Before adding temple data:

- ☐ Temple name verified from official website
- ☐ Official website URL tested and working
- ☐ No spaces or typos in URL
- ☐ HTTPS protocol used
- ☐ Website loads successfully
- ☐ Website is official (not third-party)
- ☐ Verification date documented
- ☐ Alternate URLs documented if available
